FAQ

虚拟路由器绑定外部IP地址后，虚拟机就能上公网了吗？

不一定。外部IP地址相对于私有网络而言，并不一定是公网IP。此外，虚拟机使用外部IP能否上网还跟网络规划有关。

虚拟机绑定了外部IP之后，为什么我无法通过 IP SSH 登录，也不能 ping 通该 IP ？

首先，您需要配置虚拟机的防火墙规则，如果是同一子网的不同虚拟机，您需要两台都配置防火墙规则；其次，您需要配置将虚拟机网络加入虚拟防火墙，并给虚拟路由器绑定外部IP;最后，您需要为虚拟路由器添加路由规则。

防火墙配置了IP规则后，还需要配置TCP，UDP或ICMP规则吗？

不需要。TCP，UDP，ICMP都是封装在IP协议之上，配置IP规则后则无需再配置上述协议。

网络映射配置了IP规则后，还需要配置TCP，UDP或ICMP规则吗？

不需要。TCP，UDP，ICMP都是封装在IP协议之上，配置IP规则后则无需再配置上述协议。

虚拟机密码忘了，怎么办？

您可以将虚拟机关闭，并通过管理系统功能直接修改虚拟机密码。（注：windows的虚拟机使用的镜像必须由管理员设置过。）

为什么我的 windows 主机远程桌面连接不上？

首先， 在 Windows 主机里打开RDP服务：

  1. 使用“虚拟终端”登陆 Windows 主机，“虚拟终端”就是主机 ID 旁边的 显示器形状的小图标，点击即可以弹出 Windows 桌面;
  2. 右键点击“我的电脑” -> “属性” -> “远程” -> 选择”启用这台机器上的远程桌面”。
然后您需要手动为主机的防火墙添加一条 接受 tcp 3389 端口 的下行规则， 点击 更新规则 使其应用到主机。
